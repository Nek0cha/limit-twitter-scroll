# Twitter無限スクロール防止拡張機能

X（Twitter）のタイムラインを無限スクロールしてしまう行動に対して、「ちょっとした摩擦」を与えることで自然にスクロールを止めるきっかけを作るChrome拡張機能です。

強制的にスクロールを止める・時間制限をかけるのではなく、「スクロールゲージ」というメタファーを使い、スクロールするたびにゲージが溜まっていく → 満タンになるとスクロールをロックして確認を挟む → 「続ける」を選ぶと次からゲージが溜まりにやすくなる、という体験を作ります。

## 機能

- 画面上部に半透明の通行料ゲージバーを表示（スクロール量に応じて緑→黄→赤に変化）
- ゲージが満タンになるとスクロールをロックし、「まだ見る？」の確認ダイアログを表示
  - **続ける**: ロックを解除して閲覧を継続（次の閾値が下がり、スクロールが徐々に重くなる）
  - **休憩する**: タイムラインをぼかした状態でロックを維持し、「休憩を終える」ボタンを押すまで閲覧できない
- ブラウザセッション（またはブラウザを開いている間）が終わる、もしくは日付が変わると閾値がリセット
- 拡張機能アイコンをワンクリックすると設定画面が開き、初期閾値・減少率・「休憩する」時にぼかしてロックするかをカスタマイズ可能 (閾値は8000くらいがちょうどいいです。)

## 技術構成

- [WXT](https://wxt.dev/)（TypeScript / Manifest V3）
- Vanilla TS + Shadow DOMでオーバーレイUIを注入
- `chrome.storage.local`（設定・日次リセット判定）/ `chrome.storage.session`（ゲージの蓄積状態）

## 開発

Node.js が必要です。

```bash
npm install
npm run dev      # 開発モード（HMR付き）
npm run build    # 本番ビルド（.output/chrome-mv3 に出力）
npm run compile  # 型チェックのみ
```

## インストール方法（Releaseからパッケージ化されていない拡張機能として読み込む）

Chrome Web Storeには公開していないため、[Releases](../../releases) から配布用のzipをダウンロードして読み込んでください。Chromium系ブラウザ（Chrome、Edge、Braveなど）であれば動作します。

1. [Releases](../../releases) ページから最新の `.zip` をダウンロードし、任意のフォルダに解凍する
2. ブラウザで拡張機能の管理ページを開く（Chromeなら `chrome://extensions`、Edgeなら `edge://extensions`）
3. 右上の「デベロッパーモード」をONにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、手順1で解凍してできたフォルダ（`manifest.json` が直下にあるフォルダ）を選択する
5. 拡張機能一覧に「通行料スクロール」が表示されれば完了

コードを変更しながら試したい場合は、`npm run build` で生成される `.output/chrome-mv3` フォルダを同様に読み込んでください（`npm run dev` を使うと変更が自動的に反映されます）。
